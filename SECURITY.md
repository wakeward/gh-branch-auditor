# Security Policy

## Supported Versions

I support the latest major release of `gh-branch-auditor`.

| Version | Supported          |
| ------- | ------------------ |
| Latest  | :white_check_mark: |
| < Latest| :x:                |

## Reporting a Vulnerability

I take the security of `gh-branch-auditor` seriously. If you believe you have found a security vulnerability, please report it to me as described below.

### How to Report

Please do **not** report security vulnerabilities through public GitHub issues.

Instead, please report them via:
- **GitHub Security Advisories**: Use the "Report a vulnerability" button on the [Security tab](https://github.com/wakeward/gh-branch-auditor/security/advisories).

### Response Timeline

I will acknowledge your report within **48 hours** and aim to provide a resolution or mitigation plan within **5 business days**.

### Scope

**In Scope**:
- The `gh-branch-auditor` CLI binary.
- The build and release pipeline integrity.

**Out of Scope**:
- Vulnerabilities in third-party libraries (unless actionable by me updating them).
- GitHub API behavior itself.
